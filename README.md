# Password Strength Analyser

A Java desktop application that analyzes password strength in real-time and provides instant 
feedback to help users create secure passwords.

## Features

- **Real-Time Analysis**: Password strength analyzed as you type
- **Five Security Criteria**: Length, Uppercase, Lowercase, Digits, Special Characters
- **Visual Feedback**: Color-coded progress bar (Red → Orange → Yellow-Green → Green)
- **Show/Hide Password**: Toggle visibility with checkbox
- **Clear Button**: Reset all fields instantly

## Scoring System

| Score | Rating | Color |
|-------|--------|-------|
| 0-1 | Weak | Red |
| 2-3 | Moderate | Orange |
| 4 | Strong | Yellow-Green |
| 5 | Very Strong | Green |

## System Requirements

- Java 8 or higher
- IntelliJ IDEA, NetBeans, or Eclipse
- Minimum screen resolution: 840x520 pixels

## How to Run

1. Clone the repository
2. Open project in your IDE
3. Navigate to `src/PasswordStrengthAnalyser.java`
4. Click Run (or press Shift+F10)
